### Added

- New `transforms` module for providing easy pipeline-like transformations to documents.
- New `ocr` module for providing easy OCR utilities for converting files to documents.
- Langchain - support `transforms` with `langchain`.
- Azure - support `ocr` with Azure document intelligence.
- Llama Index - Support `ocr` with `liteparse` by Llama Index.
